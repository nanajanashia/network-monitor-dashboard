# ქსელური ნაკადის მონიტორინგის მონაცემები

ქსელის მონიტორინგის მონაცემები, რომელიც აჩვენებს პაკეტების ინფორმაციას და VirusTotal-ის ანალიზის შედეგებს.

## წინაპირობები

- Go 1.20+
- PostgreSQL
- `packet_info` ცხრილი ბაზაში

## კონფიგურაცია

დააკოპირეთ `.env.example` ფაილი `.env`-ად და შეავსეთ თქვენი მონაცემებით:

```bash
cp .env.example .env
```

## გაშვება

```bash
go run main.go
```

შემდეგ გახსენით ბრაუზერში: http://localhost:8080

## API

- `GET /` - მთავარი დეშბორდი
- `GET /api/packets?after_id=12` - აბრუნებს პაკეტებს მითითებული ID-ს შემდეგ
