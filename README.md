# ქსელური ნაკადის მონიტორინგის სტატისტიკა

ქსელის მონიტორინგის სტატისტიკა, რომელიც აჩვენებს პაკეტების ინფორმაციას და VirusTotal-ის ანალიზის შედეგებს რეალურ დროში.

## რას აკეთებს

- PostgreSQL ბაზიდან კითხულობს `packet_info` ცხრილს
- აჩვენებს ბოლო 1000 ჩანაწერს (უახლესი პირველი)
- ავტომატურად განახლდება ყოველ 1 წამში ახალი ჩანაწერების გამოჩენისას
- ფერებით გამოყოფს საშიშ (malicious) და საეჭვო (suspicious) IP მისამართებს

## წინაპირობები

- Go 1.20+
- PostgreSQL
- `packet_info` ცხრილი ბაზაში

## კონფიგურაცია

დააკოპირეთ `.env.example` ფაილი `.env`-ად და შეავსეთ თქვენი მონაცემებით:

```bash
cp .env.example .env
```

## გაშვება

```bash
# დამოკიდებულებების დაყენება
go mod tidy

# კომპილაცია
go build -o network-monitor-dashboard

# გაშვება
./network-monitor-dashboard
```

შემდეგ გახსენით ბრაუზერში: http://localhost:8080

## API

- `GET /` - მთავარი დეშბორდი
- `GET /api/packets?after_id=12` - აბრუნებს პაკეტებს მითითებული ID-ს შემდეგ
